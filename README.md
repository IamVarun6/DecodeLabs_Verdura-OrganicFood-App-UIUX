## Conclusion

The Verdura Organic Food App was designed to create a simple, modern, and engaging shopping experience for users looking to purchase fresh organic products. Through a clean visual hierarchy, intuitive navigation, consistent design system, and interactive prototype, this project demonstrates key UI/UX design principles such as usability, accessibility, and user-centered design.

This project helped strengthen my skills in wireframing, visual design, component creation, prototyping, and design system development using Figma. It represents my learning journey in UI/UX design and showcases my ability to transform user requirements into a functional and visually appealing mobile application experience.
